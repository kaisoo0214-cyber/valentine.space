# 经历资料文档

请将你的详细经历文档（Word、PDF、Markdown等格式）放入此文件夹。

## 文档结构建议

### 1. 实习经历 (Internships)
- 公司名称
- 职位
- 时间
- 主要职责
- 量化成果

### 2. 项目经历 (Projects)
- 项目名称
- 时间
- 项目背景
- 你的角色
- 核心行动
- 项目成果

### 3. 校园经历 (In Campus)
- 竞赛获奖
- 学生组织
- 社团活动
- 其他校内项目

## 注意事项

- 请按时间倒序排列
- 突出"做了什么 + 产生了什么结果"
- 尽量使用数据量化成果
- 避免三个板块之间的信息重复
